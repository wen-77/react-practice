# React Practice

這是使用 React 搭配 Webpack 開發的練習專案，主要用於練習元件拆解、API 串接和狀態管理。

## 線上展示
*  成果展示網站：[React Practice](https://wen-77.github.io/react-practice/)

## 使用技術
* **前端框架**：React 19
* **建構工具**：Webpack 5
* **樣式處理**：Tailwind CSS

## Getting Started

請依照以下步驟在你的電腦上執行此專案：

1. **複製此專案到本地**
   ```bash
   git clone https://github.com/wen-77/react-practice.git
   cd react-practice

2. **安裝必要套件**
   ```bash
   npm install

3. **啟動伺服器**
   ```bash
   npm run dev

4. **啟動伺服器**
   ```bash
   npm run build
